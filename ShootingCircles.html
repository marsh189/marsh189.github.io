Shooting Circles

One of my duties as a Lead Instructor at MVCode is to create new lessons for the kids to follow. I decided to make a simplistic art style 2D Top Down shooter. The goal is to destroy the waves of enemies as they spawn in. As this was target towards kids who have little experience using Unity, the game was meant to be easy to play and code so that they can learn new concepts.

It started as a inroduction to Unity's UI sytem by making a menu screen that will load in your game. The kids then learned how to make a player object that will take input from the user to move around. I added some projectiles that can be shot by clicking the mouse and some enemies that you must destroy. The next step was adding a level manager that told the game what is currently happening (i.e. playing, game is over, game is paused, etc.). That was used to tell the level when to spawn enemies and how many.

My goal for this game is to make a more advanced version. I want to add upgrades to the guns, different base character with different speeds and health, and multiple types of enemeis.
