Battleship

This was a class project for a Data Communications/Computer Networks class. In a team of three people, we constructed a networked version of the classic Battleship game. Using Java GUI, we made a very simple board layout where the players can setup their boards, connect to the server, and play against a friend.

To Play The Game

Visit my Github page here to download the project.This is a console application which needs Java to run. The instructions on how to use the game is in the README on Github.

My Key Contributions

For this project, my job was to design the GUI. To do this I had to learn how to use Java's GUI system. After I had the board set up and my teammates had the server and clients talking to each other, I then made the checks to see if the player hit the target or not.
