The House on Haunted Hollow

This was a Freelance project where a friend and I were hired to develop an application for a children's book using Unity. There were many features that were to be added other than the book. These include mini games, picture animations, narrations, and background music. The application was equipped with a settings menu where the user can change values such as which narrator to use, how loud the background music should be, and more. The House On Haunted Hollow free version can be found on the Google Play store here.
